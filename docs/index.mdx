## 여기서 뭘 할 수 있어?

GrowFit Docs는 아래 내용을 빠르게 찾을 수 있게 정리한 문서야.

- **실습(Practice)**: 모델/파라미터/프롬프트/에이전트/RAG를 바꿔가며 비교하는 방법
- **지식베이스(RAG)**: 청크/오버랩/top-k/threshold/rerank 설정과 결과 해석
- **API**: FastAPI 기반 엔드포인트/요청·응답 예시/에러 케이스
- **운영/비용**: 토큰/지연시간/비용 지표를 교육에 연결하는 방법
